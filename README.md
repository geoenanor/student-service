# student-service
API-REST con swagger

Para swagger:

	1) Ejecutar como aplicacion java StudentServicesApplication
	2) Desde el navegador  http://localhost:8888/swagger-ui.html
	
El numero de puerto está configurado en application.properties:

	server.port=8888
